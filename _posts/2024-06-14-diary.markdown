# 2024-06-14
회사 프로젝트 base 서비스의 아키텍처
### Hexagonal Architecture
```agsl
com.example
 ├── application
 │    ├── domain
 │    │    ├── entity
 │    │    │    └── Product.kt
 │    │    ├── valueobject
 │    │    ├── service
 │    │    │    └── ProductService.kt
 │    │    └── repository
 │    │         └── ProductRepository.kt
 │    ├── usecase
 │    │    └── GetProductUseCase.kt
 │    └── service
 ├── adapters
 │    ├── inbound
 │    │    ├── controller
 │    │    │    └── ProductController.kt
 │    │    └── listener
 │    └── outbound
 │         ├── repository
 │         │    └── ProductRepositoryImpl.kt
 │         └── client
 └── configuration
      ├── spring
      │    └── AppConfig.kt
      └── bean

```

### Process example
user -> adapter(controller) -> port(useCase) -> domain & application(service) -> port -> adapter -> repository -> DB
